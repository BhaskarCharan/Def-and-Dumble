SIGN LANGUAGE DETECTION

A Brief Introduction

What is sign language detection? Basically the concept of our idea is to understand the sign language of def and dumb people . There are millions of def and dumb in India and other countries who are still struggling to communicate to other normal people with the sign language. The application which we are going to develop converts sign language to normal English in voice and text and in turn takes our text or voice command to sign language. So that it will be helpful for both.

IDEA

Idea Phase 1:

creating a web app which converts sign language(ASL) to normal English voice and text using and machine learning , html ,CSS, java script and other frame works used for web development.

Idea Phase 2:

There are many types of sign languages respective to their country they live and language in which the local people speak so we are going to include ISL (Indian Sign Language) and other sign languages in this web app.

Idea Phase 3:

Converting English voice and text to ASL . A cartoon bot will represent the ASL in the web application using same programming languages but we use UNITY software to create a cartoon character which present the sign language.

Idea Phase 4:

The algorithm can in turn learn a sign language and can create a sign language . This can be possible using Artificial intelligence.
